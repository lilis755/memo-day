# Memo Day

Memo Day 是一个本地运行的个人工作日程工具，用时间轴记录每天的事项、备注和回看线索。

它解决的不是“再做一个待办列表”，而是把一天中的任务、会议、课程和临时事项沉淀成可回顾的工作轨迹，方便写周报、复盘和追踪当天产出。

## 核心功能

1. 按日期查看每日时间轴。
2. 新增事项并设置开始时间、持续时长和颜色分类。
3. 点击时间轴空白区域快速选择时间段。
4. 拖动调整事项时长。
5. 为每个事项保存备注。
6. 支持日历切换历史日期。
7. 通过本地 Node 服务保存到 `events.json`。
8. 浏览器 `localStorage` 作为兜底备份。
9. 支持 PWA 离线壳和 macOS 本地 App 包装。

## 启动

```sh
npm start
```

然后打开：

```txt
http://127.0.0.1:4173/index.html
```

也可以双击：

```txt
Memo Day.command
```

```txt

```

页面、样式、脚本、图标和数据都在这个文件夹里，适合上传到 GitHub/GitLab 后继续迭代。
